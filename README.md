# Zeen, text-based dungeon crawler
this is a simple dungeon crawler, where you're forced to go through a dangerous dungeon, filled to the brim with TWO whole different kind of enemies! defeat foes to level up and become stronger, with the whole goal of escaping the dungeon.

## How to Play

1. Clone this repository:
   git clone https://github.com/yourusername/dungeon-crawler.git

2. Navigate to the game folder.

3. Run the game:
   python game.py

## Features
- turn-based combat
- Inventory system
- enemies scale with your level
- Interactive UI
